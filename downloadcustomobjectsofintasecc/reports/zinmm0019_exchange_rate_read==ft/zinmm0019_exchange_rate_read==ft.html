FORM COMPARE_DATA
                  TABLES DIFFERENCES STRUCTURE UNTERSCHIED
                  USING FIELD
                  CHANGING RETCODE.
 DATA : SUBRC LIKE SY-SUBRC.
 field-symbols: <l_f1> type x.
 field-symbols: <l_f2> type x.
 RETCODE = '='.
 CLEAR DIFFERENCES.
 CASE FIELD.
WHEN 'E_FINAL'.
DO.
READ TABLE %_OE_FINAL INDEX SY-INDEX.
SUBRC = SY-SUBRC.
READ TABLE %_VE_FINAL INDEX SY-INDEX.
IF SY-SUBRC NE 0 AND SUBRC NE 0.
EXIT.
ELSEIF SY-SUBRC NE 0.
RETCODE = SPACE.
MOVE FIELD TO DIFFERENCES-PARAMETER.
assign %_OE_FINAL to <l_f1> casting.
assign differences-neu to <l_f2> casting.
<l_f2> = <l_f1>.
APPEND DIFFERENCES.
EXIT.
ELSEIF SUBRC NE 0.
RETCODE = SPACE.
MOVE FIELD TO DIFFERENCES-PARAMETER.
assign %_VE_FINAL to <l_f1> casting.
assign differences-alt to <l_f2> casting.
<l_f2> = <l_f1>.
APPEND DIFFERENCES.
EXIT.
ELSEIF %_VE_FINAL
NE %_OE_FINAL.
RETCODE = SPACE.
MOVE FIELD TO DIFFERENCES-PARAMETER.
assign %_VE_FINAL to <l_f1> casting.
assign differences-alt to <l_f2> casting.
<l_f2> = <l_f1>.
assign %_OE_FINAL to <l_f1> casting.
assign differences-neu to <l_f2> casting.
<l_f2> = <l_f1>.
APPEND DIFFERENCES.
EXIT.
ENDIF.
ENDDO.
WHEN 'EXCEPTION'.
IF VEXCEPTION NE OEXCEPTION.
